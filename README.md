# LogReport

A small tool for reading a log file and generating an XML report.

## What was used in this project:
* Gradle
* Spring Spring Boot
* JAXB API
* Lombok
* Mongo
* JUnit
* Mockito

## How to use the project

* The application will print on the console a message to enter the file name.

```bash
$ gradlew bootRun

```
